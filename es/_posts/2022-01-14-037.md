| Comando           | Descripción                                      |
| ----------------- | ------------------------------------------------- |
| `reset`         | Comando que restaura a un estado anterior         |
| `HEAD`          | Puntero que guarda el estado actual               |
| `^`             | El circunflejo es un atajo para el último commit |
